Maîtrise avancée de Java pour le développement d'applications robustes et scalables.

- Programmation orientée objet (POO)
- Java SE et Java EE
- Frameworks : Spring, Spring Boot, Hibernate
- Gestion des threads et concurrence
- Tests unitaires avec JUnit
- Maven et Gradle pour la gestion de dépendances

Java est mon langage de prédilection pour les applications backend nécessitant performance et fiabilité.
